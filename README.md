# PLACEHOLDER NAME

### By Jidan Ahmed

### Project Description:

This is a multiplayer game where players exist on the same map and can see each other's locations. Players avoid a monster to collect points.

### Instructions:
Both players compile with make or make compile

Host player runs ./server with the argument being the map file. (If not specified, a default map will be used.)

Guest player runs ./client with the argument being the host's IP. (If not specified, a default IP will be used.)

Move with WASD.

Collect p'o'ints and avoid the 'M'onster to get a high score.
